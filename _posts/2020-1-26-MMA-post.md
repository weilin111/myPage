# 排版

        


esc  

        你可以通过它得到 任意希腊字母 和数学运算符号  

        esc + alpha + esc = α

        esc + pd + esc  =  \partial 真partial

        ....




Ctrl + 
        
        通过它可以得到  结构上的 下标 上标  分式     


        Ctrl + 6  = 右上标
        
        Ctrl + _  = 右下标

        Ctrl + 2  = 根号

        Ctrl + /  = 分式



alt +  
        通过它可以得到  不同的block 比如代码block

        alt + 7  代码block
